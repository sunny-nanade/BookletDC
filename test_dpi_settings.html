DPI Settings Analysis for 1440p Document Scanning

Current Settings Analysis

Image Resolution: 1440 × 2560 pixels

Previous DPI: 300 (default)

New DPI: 400 (33% increase)

Target A4 Coverage: 98% (reduces white space)

DPI Impact on PDF Quality

At 300 DPI:

  • PDF page size: 121.92 × 216.75 mm
  • Print quality: Good for most documents
  • File size: Moderate

At 400 DPI:

  • PDF page size: 91.44 × 162.56 mm
  • Print quality: Excellent for text and fine details
  • File size: Higher, but better quality
  • Document fills more of the A4 page (less white space)

Expected Improvements

  • Reduced Empty Space: Higher DPI makes the document content larger relative to the page
  • Better Text Quality: 400 DPI provides sharper text rendering
  • Optimized Coverage: 98% A4 coverage ensures maximum page utilization
  • Professional Quality: 400 DPI is ideal for business documents

Technical Details

1440p Image at 400 DPI:

  • Width: 1440px ÷ 400 DPI = 3.6 inches = 91.44 mm
  • Height: 2560px ÷ 400 DPI = 6.4 inches = 162.56 mm
  • A4 width utilization: 91.44mm ÷ 210mm = 43.5% → scaled to 98% = larger document

Recommendations

  1. Test the new 400 DPI setting - Should reduce empty space significantly
  2. If still too much space: Consider 450-500 DPI for maximum page filling
  3. For ultimate quality: Use 600 DPI (professional printing standard)
  4. Balance file size vs quality based on your needs
